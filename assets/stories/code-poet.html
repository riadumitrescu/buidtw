The Code Poet

by Sarah Chen

About the Creator

Sarah Chen is a full-stack developer and poet who believes in the intersection of technology and art. Her work explores the beauty of code as a form of expression, translating complex algorithms into elegant, readable poetry.

The Design

The hoodie features a minimalist design that incorporates elegant code snippets from Sarah's favorite algorithms. The front displays a poetic implementation of a binary search tree, while the back showcases a haiku-inspired sorting algorithm. The design uses monospace typography and subtle geometric patterns to create a balance between technical precision and artistic expression.
